A contact form example using the Atlas Content Modeler crud api.

== Description ==

An example plugin using the Atlas Content Modeler crud api to collect data on the front-end from a contact form and store the data as an entry model.

This will also use the crud api validation to enforce correct data.

Do not use this as your contact form. As it provides no spam protection.

== Installation ==

Ensure the Atlas Content Modeler plugin is installed and enabled.

Use the WordPress Plugin > Install New and upload the zip file.
